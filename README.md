# StarfighterPatrol

* bach.py - provides musical tones
* starfighter.py - 1D starfighter patrol game. Destroy asteroid threats.
* starfighter.js - Javascript version Makecode link: https://makecode.com/_RocF4AHbY19R

  A - fire laser
  B - reverse direction
  A4 - hyperspace
  A3 - restart game
  A7 - show last score
